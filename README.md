# Simple Calculator

##  Project Description
This is a command-line based Simple Calculator developed using Python. It allows users to perform basic arithmetic operations through a menu-driven interface.

##  Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Handles invalid number inputs using exception handling
- Allows users to perform multiple calculations using a loop

##  Technologies Used
- Python
- Conditional Statements (if-elif-else)
- While Loop
- Exception Handling

##  How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open the project in VS Code.
4. Run the following command:

```bash
python main.py
```
